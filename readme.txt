== Description ==
LR Portfolio & Gallery is an amazing and powerful WordPress portfolio plugin designed to help you display your portfolio projects. 

You can quickly add portfolio to your portfolio(s), display them anywhere with a easy shortcode.
it’s totally free.

For more features Please contact us at [Logicrays](http://www.logicrays.com/contact-us-logicrays-india/)
If you like this plugin Please rate us :)

== Plugin Features ==
Responsive Portfolio plugin
All modren browser support
works with anytype of theme
Easy to customize for developeres
Easy to configure

== Shortcode ==
`
[LRPORTFOLIOS]
`
== Installation ==

1. Upload Zip file lr-portfolio to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
